### Installation

#### For users

```shell
# e.g) with poetry
poetry add git+https://github.com/tier4/t4-devkit.git
```

#### For developers

```shell
git clone git@github.com:tier4/t4-devkit.git
cd t4-devkit
poetry install
```

> [!NOTE]
> We are planning to publish `t4-devkit` to PyPI in the future release.
